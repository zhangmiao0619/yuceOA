# OA 系统测试文档

**项目名称：** 武汉宇测科技 OA 办公系统
**测试范围：** 功能测试 + 系统测试
**最近更新：** 2026-05-12
**依据版本：** OA 系统情况说明 v1.0

---

## 一、测试环境

| 项目 | 说明 |
|------|------|
| 前端地址 | http://localhost:5173 |
| 后端 API | http://localhost:3000 |
| 技术栈 | React 18 + TypeScript + Vite / Hono (Node.js) + PostgreSQL |
| 认证方式 | JWT |
| 登录方式 | 用户名/密码（企业微信登录已禁用） |

### 1.1 前置条件

- 后端服务运行于 3000 端口
- 前端服务运行于 5173 端口
- PostgreSQL 数据库运行于 5432 端口
- 测试账号：系统初始化时创建的管理员账号
